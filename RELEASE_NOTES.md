# test Location 1.17

This release updates test Location to version 1.17.

Highlights:

- Corrected the Add Usage Time page card titles to Add One Month and Add One Year.
- Restored the blue and orange action buttons to their original Subscribe Now text.
- Retains the 1.15 official website footer.
- Published as a forced upgrade release.

Installation:

1. Download `test-location-1.17.apk` from this release.
2. Install it on an Android device.
3. Enable Developer Options.
4. Set `test Location` as the mock-location app.
5. Open the app and start testing coordinates.

Use this software only for development, debugging, QA, and lawful testing.
